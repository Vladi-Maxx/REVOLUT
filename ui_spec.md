# Спецификация за потребителския интерфейс

Създайте интерфейс, който включва:
1. Заглавна секция с обобщени суми
2. Филтри за:
   - Времеви период
   - Валута
   - Тип транзакция
3. Таблица, показваща:
   - Търговец
   - Брой транзакции
   - Обща сума
   - Средна сума
4. Графика, визуализираща разпределението на разходите по търговци
